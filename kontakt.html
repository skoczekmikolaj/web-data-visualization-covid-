Nazywam się Mikołaj Skoczek, jestem studentem 3 roku Informatyki i Ekonometrii. Programuję w językach takich, jak C#, Python czy PHP. Do moich umiejętności należy także znajomość baz danych w standardach SQL Server oraz MySQL, tworzenie stron internetowych przy wykorzystaniu HTML, CSS, Javascript i Bootstrap. Podczas pracy z pakietem .NET poznałem WPF oraz Entity Framework. Znam też formaty XML i JSON. Ponadto rozumiem podstawy Statystyki, Ekonometrii czy Ekonomii, a także posługuję się bezproblemowo pakietem Office. Umiem też korzystać z systemów kontroli wersji GIT i GITHUB.

Odbyłem miesięczną praktykę w Krajowej Izbie Rozliczeniowej. Jestem osobą ambitną, pracowitą i szybko się uczącą.

W wolnym czasie najchętniej zajmuję się produkcją muzyki i grą na instrumentach klawiszowych. Lubię także aktywnie spędzać czas, uprawiając sporty, takie jak pływanie, piłkę nożną czy ćwicząc na siłowni. Ważną częścią mojego życia jest także ratownictwo - w przeszłości pracowałem jako ratownik wodny.
